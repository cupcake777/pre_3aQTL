# compare

`compare/` covers two comparison layers:

1. overlap, effect-direction consistency, and sharing across developmental stages;
2. cross-dataset validation against external public 3'aQTL reference sets.

## Primary script

- [`01_combined_analysis.R`](/mnt/share_group_folder/tmp/work/Pre_3aQTL/Three_stages/03_QTL/compare/01_combined_analysis.R)
  Integrates within-study stage comparisons and external reference comparisons.
